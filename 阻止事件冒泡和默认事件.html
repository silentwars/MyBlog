阻止事件冒泡和默认事件

javascript中

针对不同的浏览器，主要分为IE和非IE两种进行区分，同时也有两种写法。

          //阻止冒泡事件
                    function stopBubble(e){
  	                        if(e&& e.stopPropagation){
   		                      //非IE
   		                      e.stopPropagation();
   	                         }else{
   		                      //IE
   		                        window.event.cancelBubble=true;
   	                         }
                            }
          
                      function stopBubble(e){
   	                              e=e||window.event;
   	                            if(e.stopPropagation){
   		                  e.stopPropagation();
   	                            }else{
   		                            e.cancelBubble=true;
   	                            }
                               }

                     //阻止默认事件
                     function stopDefault(e){
             	             e=e||window.event;
   	                       if(e.stopDefault){
   		                       e.stopDefault();
   	                       }else{
   		                       e.returnValue=false;
   		                       return false;
   	                       }
                          }

                     function stopDefault(e){
   	                       if(e&&e.stopDefault){
   		                       e.stopDefault();
   	                       }else{
   		                       e.returnValue=false;
   		                       return false;
   	                       }
                          }

jQuery中

jQuery 提供了两种方式来阻止事件冒泡。

方式一：event.stopPropagation();

                      $("#div1").mousedown(function(event){
                            event.stopPropagation();
                        });

方式二：return false;

                      $("#div1").mousedown(function(event){
                            return false;
                       });

但是这两种方式是有区别的。return false 不仅阻止了事件往上冒泡，而且阻止了事件本身。event.stopPropagation() 则只阻止事件往上冒泡，不阻止事件本身。

jQuery阻止默认事件

                     $("#div1").mousedown(function(event){event.preventDefault();});
